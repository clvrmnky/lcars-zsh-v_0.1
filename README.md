# LCARS Zsh Theme (TNG-inspired)

Pure Zsh prompt theme designed for macOS Terminal.app (and iTerm2). No dependencies.

## Install
```sh
git clone https://github.com/clvrmnky/lcars-zsh-v_0.1.git
cd lcars-zsh-v_0.1
./install.sh
```

## Features

- Three-line LCARS prompt with color-coded panels
- Git branch and dirty-state indicator
- Input color state tracking (idle vs active)
- Enterprise NCC-1701-D ASCII art on boot
- Built-in LCARS commands

## Commands

| Command   | Description           |
|-----------|-----------------------|
| `info`    | System diagnostics    |
| `scan`    | Live system scan      |
| `nav`     | Navigation status     |
| `ops`     | Git operations        |
| `bridge`  | Full bridge readout   |
| `cmds`    | List all commands     |
